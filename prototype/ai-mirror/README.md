# AI Mirror 프로토타입 v0.2

> 플랜의 **실험 6 "거울을 작동시키다"** 산출물.
> 자동 인식 → Claude 비전 해석 → 배경 변환 + 추천 목록까지의 파이프라인.

## 무엇을 하나 (v0.2)

1. 동의 화면 (실험 7 '신뢰의 조건' 반영 — 사진은 해석 후 즉시 폐기, 어디에도 저장 안 함)
2. 웹캠이 거울처럼 켜짐 (좌우 반전)
3. **거울 앞에 2초간 멈춰 서면 자동으로 촬영** (버튼 필요 없음, 진행 링 표시)
4. Claude가 사진을 보고:
   - **근거 있는 시** 한 문장 (관찰한 사실 1개 = 앵커 + 무드 단어)
   - 카탈로그 7개 모델 중 **무드에 어울리는 제품 3개** 선택
5. 결과 연출:
   - 사람은 그대로, **배경만 무드에 맞게 변환** (사람/배경 분리 합성)
   - **왼쪽 위에 추천 아이템 목록** (카탈로그 카드 스타일, 이유 없이 아이템만)
   - 해석까지 걸린 시간 표시 (RQ4 대기 시간 측정용)
6. 사람이 자리를 뜨면 자동으로 처음 화면으로 복귀

### 기술 구성

| 역할 | 도구 | 실행 위치 |
|---|---|---|
| 얼굴 감지 + 정지 판정 | MediaPipe Face Detector (무료) | 브라우저 |
| 사람/배경 분리 | MediaPipe Selfie Segmenter (무료) | 브라우저 |
| 무드 해석 + 제품 추천 | Claude Opus 4.8 (비전) | 서버 → Anthropic API |
| 무드 배경 | 사전 제작 이미지 5종 (없으면 그라데이션 대체) | 브라우저 |

- MediaPipe는 처음 로드할 때 인터넷이 필요합니다. 로드 실패 시 수동 [읽기] 버튼 모드로 자동 전환됩니다.
- 추천은 `public/catalog.json`(exp02 카탈로그 기반) 안에서만 고르도록 **스키마로 강제**되어 있어, 없는 제품을 지어내는 환각이 원천적으로 불가능합니다.

## 실행 방법

터미널에서 이 폴더로 이동한 뒤:

```powershell
cd "prototype\ai-mirror"
npm install        # 처음 한 번만
npm start
```

브라우저에서 **http://localhost:3737** 을 엽니다.

### 데모 모드 vs 라이브 모드

| 모드 | 조건 | 비용 |
|---|---|---|
| **데모** | API 키가 없을 때 자동 전환 (또는 아래처럼 강제 실행) | 0원 — 준비된 예시 결과를 보여줌 |
| **라이브** | API 키가 있을 때 | 사진 1장당 약 수십 원 수준 (Claude Opus 4.8 기준) |

데모 모드 강제 실행 (API 호출 없이 화면 흐름만 테스트할 때):

```powershell
$env:DEMO="1"; npm start
```

라이브 모드 설정 — Anthropic API 키를 환경변수로 넣고 실행:

```powershell
$env:ANTHROPIC_API_KEY="sk-ant-..."; npm start
```

> API 키는 서버(server.js) 안에서만 사용되고 브라우저로는 전달되지 않습니다.

### 배경/제품 이미지 넣기

`이미지-생성-프롬프트.md`의 프롬프트로 이미지를 만들어 아래 위치에 저장하면 자동 반영됩니다:

- 무드 배경 5장 → `public/assets/backgrounds/warm.jpg` 등
- 제품 이미지 7장 → `public/assets/products/papas-01.png` 등

이미지가 없어도 작동합니다 (배경은 그라데이션, 제품은 모노그램으로 대체).

### 조절할 수 있는 값

- `server.js` 상단: `MODEL`(모델), `EFFORT`(해석 깊이/속도)
- `public/app.js` 상단: `HOLD_MS`(정지 판정 시간), `MOVE_TOLERANCE`(움직임 허용치), `ABSENT_RESET_MS`(자리 비움 복귀 시간)

## 개인정보 처리 (실험 7 연결)

- 얼굴 감지·사람/배경 분리는 **전부 브라우저 안에서** 처리 (밖으로 안 나감)
- 촬영된 사진 1장만 해석을 위해 서버 → Anthropic API로 한 번 전달
- 디스크 저장 없음, 로그에 사진 데이터 기록 없음

## 실험 기록 (측정)

  시각 / 모드 / 전체 소요 초 / API 소요 초 / 해석·추천 결과 전체
