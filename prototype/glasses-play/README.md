# Glasses Play — 모션트래킹 선글라스 장난감

> 주말 실험. 웹캠 앞에서 손동작으로 가상 선글라스를 접었다 폈다 하고, 집어서 착용까지.
> API 호출 없음 — 손·얼굴 추적(MediaPipe)이 전부 브라우저 안에서 실행됩니다. 비용 0원.

## 손동작

| 동작 | 효과 |
|---|---|
| 🙌 두 손을 모으고 / 벌리고 | **코 브릿지**가 접힘 / 펴짐 |
| ✌️ 한 손 검지·약지를 오므리고 / 펴고 | **다리(템플)**가 접힘 / 펴짐 |
| 🤏 엄지+검지로 집어서 얼굴로 가져가 놓기 | **착용** (착용 중 다시 집으면 벗기) |

착용하면 선글라스가 얼굴을 따라다니고(고개 기울임·거리 변화 포함), 자동으로 활짝 펴집니다.

## 실행

```powershell
cd "prototype\glasses-play"
node server.js
```

브라우저에서 **http://localhost:3838** 을 엽니다. (설치할 것 없음 — `npm install` 불필요)

## 인식 팁

- **손바닥이 카메라를 향해야** 손 인식이 잘 됩니다 (손등은 잘 안 잡힘 — 모델 특성)
- 집기는 엄지+검지를 **확실히 붙인 채 선글라스 위에서 잠깐(0.2초) 유지**해야 잡힙니다 (오발동 방지용)

## 조절할 수 있는 값 (public/app.js 상단)

- `PINCH_ON` / `PINCH_OFF` — 집기 시작/놓기 판정 (붙임/벌림 정도)
- `NOT_PINCHING` — 이보다 벌어져야 브릿지·다리 제스처로 인정 (집기와 충돌 방지)
- `GRAB_TICKS` — 잡기/놓기 확정에 필요한 연속 감지 횟수
- `BRIDGE_NEAR` / `BRIDGE_FAR` — 두 손 간격 → 브릿지 접힘 범위
- `TEMPLE_CLOSED` / `TEMPLE_OPEN` — 검지·약지 간격 → 다리 접힘 범위
- 선글라스 치수: `GAP`, `HALF_W`, `LENS_H`, `TEMPLE_L`

제스처가 너무 민감하거나 둔하면 위 값을 조금씩 바꿔 보세요.

## 구조

- 선글라스는 **코드로 그린 4조각** (앞면 절반 ×2 + 다리 ×2)
  - 브릿지 경첩: 가운데 기준으로 양쪽 앞면의 폭이 압축되며 접힘
  - 다리 경첩: 바깥 모서리 기준으로 위로 스윙하며 접힘 (펴지면 원근 단축)
- 나중에 실제 제품 사진으로 교체하려면 같은 4조각 구조의 투명 PNG가 필요
