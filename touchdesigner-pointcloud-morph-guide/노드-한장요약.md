# 노드 한 장 요약

원본 영상: https://www.youtube.com/watch?v=sE716KGZw0w

```text
[MODEL A]
PointFileIn_A → PointTransform_A → Fit(2X) → Fit(800²) → pos_A ─┐
       └──────→ PointFileSelect_A → Math(÷127) → Fit×2 → col_A   │
                                                        │
[MODEL B]                                               ├→ switch_pos → Add(noise) → POS
PointFileIn_B → PointTransform_B → Fit(2X) → Fit(800²) → pos_B ─┘
       └──────→ PointFileSelect_B → Math(÷127) → Fit×2 → col_B  ┐
                                                        ├→ switch_col → COL
                                                  col_A ┘

POS → geo_particles Instance Translate OP: r/g/b
COL → geo_particles Instance Color OP:     r/g/b/a
Add SOP(Add Points) → Convert SOP(Render as Point Sprites)
                    + Point Sprite MAT → geo_particles → render1

keyboardin1 (Key 1)
  ├→ Math(0~1 → 0~0.37) → Lag → AMP → noise1 Amplitude
  └→ Lag(.5/.2) → Count(감소 0.7, 0/1 loop) → Lag → INDEX → 두 Switch의 Index
```

반드시 지킬 것:

1. `pos_A`, `pos_B`, `col_A`, `col_B` 해상도를 똑같이 맞춘다.
2. 위치 TOP은 32-bit float RGBA를 사용한다.
3. 두 Switch의 입력 순서를 동일하게 한다.
4. `geo_particles` 안에 인스턴싱할 점 하나를 둔다.
5. 기본 렌더가 완성되기 전에는 Feedback과 MediaPipe를 연결하지 않는다.
