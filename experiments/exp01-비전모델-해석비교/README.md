# 실험 로그 — 01. 거울은 무엇을 읽는가 (비전 모델 해석 비교)

> 날짜: (실험한 날 기록)
> 연결된 리서치 질문: RQ1 — 고객이 선글라스를 쓴 순간, AI는 무엇을 '읽을' 수 있는가?
> 사용 도구: (예: GPT-5, Claude Fable 5, Gemini 3 — 실제 쓴 버전을 정확히)

---

## 1. 질문 — 무엇이 궁금했나

같은 착용 사진 한 장을 서로 다른 비전 AI에게 주면,
얼굴형·스타일·감정 무드·공간 분위기 해석이 **얼마나 같고 얼마나 다른가?**
그리고 그중 어떤 모델이 '시적 해석'(진단이 아니라 선물처럼 들리는 문장)에 더 어울리는가?

## 2. 방법 — 어떻게 실험했나

- **입력**: 선글라스를 착용한 사진 1~2장 (`inputs/` 폴더에 저장)
- **공통 프롬프트**: 세 모델에 똑같은 질문을 준다. (아래 프롬프트 초안 참고)
- **고정한 것**: 같은 사진, 같은 프롬프트
- **바꾼 것**: 모델만 (GPT / Claude / Gemini)
- **반복**: 각 모델당 최소 2회 (같은 답을 주는지 일관성 확인)

### 프롬프트 초안 (실험하며 다듬을 것)
```
이 사진 속 인물이 착용한 선글라스와 전체적인 인상을 보고,
다음을 각각 한 줄로 해석해줘:
1) 얼굴형의 인상 (판단이 아니라 느낌으로)
2) 스타일 무드
3) 표정에서 느껴지는 감정 톤
4) 이 사람에게 어울릴 법한 초현실 장면의 키워드 3개
전문 용어 없이, 짧고 감각적인 문장으로.
```
