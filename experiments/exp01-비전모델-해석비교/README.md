# 실험 로그 — 01. 거울은 무엇을 읽는가 (비전 모델 해석 비교)

> 날짜: 2026-07-11
> 연결된 리서치 질문: RQ1 — 고객이 선글라스를 쓴 순간, AI는 무엇을 '읽을' 수 있는가?
> 사용 도구: Claude Opus 4.8, GPT-5.6(sol), GPT-5.5, Gemini 3.1 Pro (각 모델 새 대화, 동일 프롬프트)
> 입력 사진: inputs/실험1- 여자.png (미러룸에서 선글라스 착용 셀카 — 우연히 프로젝트 컨셉과 겹침)

---

## 1. 질문 — 무엇이 궁금했나

같은 착용 사진 한 장을 서로 다른 비전 AI에게 주면,
얼굴형·스타일·감정 무드·공간 분위기 해석이 **얼마나 같고 얼마나 다른가?**
그리고 그중 어떤 모델이 '시적 해석'(진단이 아니라 선물처럼 들리는 문장)에 더 어울리는가?

## 2. 방법 — 어떻게 실험했나

- **입력**: 선글라스를 착용한 사진 1~2장 (`inputs/` 폴더에 저장)
- **공통 프롬프트**: 세 모델에 똑같은 질문을 준다. (아래 프롬프트 초안 참고)
- **고정한 것**: 같은 사진, 같은 프롬프트
- **바꾼 것**: 모델만 (GPT / Claude / Gemini)
- **반복**: 각 모델당 최소 2회 (같은 답을 주는지 일관성 확인)

### 프롬프트 초안 (실험하며 다듬을 것)
```
이 사진 속 인물이 착용한 선글라스와 전체적인 인상을 보고,
다음을 각각 한 줄로 해석해줘:
1) 얼굴형의 인상 (판단이 아니라 느낌으로)
2) 스타일 무드
3) 표정에서 느껴지는 감정 톤
4) 이 사람에게 어울릴 법한 초현실 장면의 키워드 3개
전문 용어 없이, 짧고 감각적인 문장으로.
```

## 3. 결과 — 무슨 일이 일어났나

원본 결과물은 `outputs/`에 모델별로 저장(claude_해석_01 / gpt_해석_01 / gpt55_해석_01 / gemini_해석_01).
아래는 각 모델의 대표 응답(1회차 기준)을 요약한 비교표.

| 모델 | 얼굴형 해석 | 스타일 무드 | 감정 톤 | 초현실 키워드 | 일관성(2회) | 시적인 정도 |
|---|---|---|---|---|---|---|
| Claude Opus 4.8 | 흐트러짐 없는 계란형, 중심 잡힘 | 검정 한 톤의 질감(결·구김) | 렌즈 뒤 냉정, 관찰하는 시선 | 거울 회랑 / 액체 금속 / **초록 어둠** | (1회) | 중~상, 질감 중심 |
| GPT-5.6 (sol) | 매끈한 타원형, 날렵 | 어둡고 미니멀한 미래적 시크 | 무심·침착, 거리감 | 무한 거울 / 검은 태양 / **크롬(금속)** | **매우 높음** (문장까지 유사) | 중, 담백·절제 |
| GPT-5.5 | 고요한 타원형, 정돈된 선 | 유리 같은 공간의 미래적 시크 | 관찰하는 차가운 침착 | 무한 거울 / **유리** 미로 / 검은 유리 정원 | 중 (뼈대 유지, 표현 확장) | **상**, 문학적 |
| Gemini 3.1 Pro | 날렵하고 차가운 도시의 곡선 | 반항적 아우라 / 퓨처리스틱 고딕 | 속내 감춘 은밀한 응시 | 네온 도시 / 액체 금속 미로 / 블랙홀 | **낮음** (형식까지 이탈, 해시태그화) | 상, 강렬·과장 |

### 관통하는 공통점
- **네 모델 모두** '무한 거울 + 금속/미로 + 무심한 냉정함'으로 수렴.
- 얼굴형은 전원 '타원형/매끈한 선'으로 일치 → 얼굴 자체 해석은 모델 간 이견이 거의 없음.

### 갈라지는 지점
- **질감 어휘**: GPT-5.6=크롬(금속), GPT-5.5=유리, Claude=액체 금속+초록빛. 같은 배경을 서로 다른 물성으로 번역.
- **일관성↔시적임 트레이드오프**: GPT-5.6(안정·담백) ↔ GPT-5.5·Gemini(변동 큼·표현 풍부). 안정성과 문학성이 반비례.
- **형식 안정성**: Gemini가 2회차에서 '한 줄 문장' 지시를 벗어나 해시태그로 이탈 → 프롬프트 구속력이 모델마다 다름.

## 4. 배운 것 — 그래서 무엇을 알게 됐나

1. **AI는 '사람'만큼이나 '공간'을 읽었다.** 네 모델의 초현실 키워드가 전부 배경(미러룸) 쪽으로 쏠렸다. 즉 이 사진에서 AI가 준 해석의 상당 부분은 인물이 아니라 환경에서 왔다. → 개인화라고 부르려면 '배경 영향'을 분리해야 한다는 걸 처음으로 확인.
2. **얼굴형 해석은 합의되지만, 무드·질감은 모델의 개성이 갈린다.** 개인화의 여지는 '사실(얼굴형)'이 아니라 '해석(질감·장면)'에 있다.
3. **일관성과 시적임은 동시에 얻기 어렵다.** 매장 경험은 매번 재현돼야 하므로 안정성이 중요한데, 가장 안정적인 GPT-5.6은 가장 담백했다. '근거 있는 시'를 하려면 안정 모델 + 시적 후처리 조합이 필요할 수 있다.

## 5. 다음 결정 — 이 결과가 다음 실험을 어떻게 바꾼다

- **exp01b(즉시 후속)**: 같은 사진에 "배경은 무시하고 인물만 해석" 조건을 추가해 재실험 → '공간 영향'을 분리 측정. RQ1의 핵심 질문이 됨.
- **exp02(이미지 생성)로 넘길 재료**: 질감이 갈린 세 방향(크롬 / 유리 / 초록빛 금속)을 각각 이미지 생성 프롬프트로 만들어 비교. "해석의 차이가 그림의 차이로 이어지는가"를 봄.
- **프롬프트 개선**: '한 줄 문장' 형식을 지키게 하려면 지시를 더 강하게(예: "반드시 완결된 한 문장, 해시태그 금지"). Gemini 이탈이 근거.
