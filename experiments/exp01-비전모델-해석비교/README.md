# 실험 로그 — 01. 거울은 무엇을 읽는가 (비전 모델 해석 비교)

> 날짜: 2026-07-11
> 연결된 리서치 질문: RQ1 — 고객이 선글라스를 쓴 순간, AI는 무엇을 '읽을' 수 있는가?
> 사용 도구: Claude Opus 4.8(도구 내), Claude Fable 5, GPT-5.6(sol), GPT-5.5, Gemini 3.1 Pro (각 모델 새 대화, 동일 프롬프트)
> 공통 조건: 챗으로 실행한 모델은 **reasoning effort를 최대로 설정**. (단, Opus 4.8은 Claude Code 도구 안에서 생성 — 아래 방법 주석 참고)
> 입력 사진: inputs/실험1- 여자.png (미러룸에서 선글라스 착용 셀카 — 우연히 프로젝트 컨셉과 겹침)

---

## 1. 질문 — 무엇이 궁금했나

같은 착용 사진 한 장을 서로 다른 비전 AI에게 주면,
얼굴형·스타일·감정 무드·공간 분위기 해석이 **얼마나 같고 얼마나 다른가?**
그리고 그중 어떤 모델이 '시적 해석'(진단이 아니라 선물처럼 들리는 문장)에 더 어울리는가?

## 2. 방법 — 어떻게 실험했나

- **입력**: 선글라스를 착용한 사진 1장 (`inputs/실험1- 여자.png`)
- **공통 프롬프트**: 모든 모델에 똑같은 질문을 준다. (아래 프롬프트 초안 참고)
- **고정한 것**: 같은 사진, 같은 프롬프트, reasoning effort 최대(챗 실행 모델)
- **바꾼 것**: 모델만 (Claude 2종 / GPT 2종 / Gemini)
- **반복**: 각 모델당 2회 (같은 답을 주는지 일관성 확인)
- **주의(방법상 한계)**: Claude Opus 4.8 데이터 포인트는 이 프로젝트를 진행하는 Claude Code 도구 안에서 생성되어, 챗 UI에서 effort를 최대로 준 다른 모델들과 실행 환경이 완전히 동일하진 않다. Claude 계열의 '챗+effort 최대' 조건은 Fable 5가 대표한다.

### 프롬프트 초안 (실험하며 다듬을 것)
```
이 사진 속 인물이 착용한 선글라스와 전체적인 인상을 보고,
다음을 각각 한 줄로 해석해줘:
1) 얼굴형의 인상 (판단이 아니라 느낌으로)
2) 스타일 무드
3) 표정에서 느껴지는 감정 톤
4) 이 사람에게 어울릴 법한 초현실 장면의 키워드 3개
전문 용어 없이, 짧고 감각적인 문장으로.
```

## 3. 결과 — 무슨 일이 일어났나

원본 결과물은 `outputs/`에 모델별로 저장(claude_해석_01 / claude-fable5_해석_01 / gpt_해석_01 / gpt55_해석_01 / gemini_해석_01).
아래는 각 모델의 대표 응답(1회차 기준)을 요약한 비교표. (5개 모델 = Claude 2종 + GPT 2종 + Gemini)

| 모델 | 얼굴형 해석 | 스타일 무드 | 감정 톤 | 초현실 키워드 | 일관성(2회) | 시적인 정도 |
|---|---|---|---|---|---|---|
| Claude Opus 4.8 (도구 내) | 흐트러짐 없는 계란형, 중심 잡힘 | 검정 한 톤의 질감(결·구김) | 렌즈 뒤 냉정, 관찰하는 시선 | 거울 회랑 / 액체 금속 / **초록 어둠** | (1회) | 중~상, 질감 중심 |
| Claude Fable 5 (챗·effort 최대) | 서늘하게 고요한 계란형 곡선 | 검정으로 눌러 담은 절제된 시크 | 자기 온도를 지키는 담담한 응시 | 거울 회랑 / **수직으로 쏟아지는 빛의 비** / 유리 도시 | **매우 높음** (모티프 유지) | **최상**, 풍부+안정 동시 |
| GPT-5.6 (sol) | 매끈한 타원형, 날렵 | 어둡고 미니멀한 미래적 시크 | 무심·침착, 거리감 | 무한 거울 / 검은 태양 / **크롬(금속)** | **매우 높음** (문장까지 유사) | 중, 담백·절제 |
| GPT-5.5 | 고요한 타원형, 정돈된 선 | 유리 같은 공간의 미래적 시크 | 관찰하는 차가운 침착 | 무한 거울 / **유리** 미로 / 검은 유리 정원 | 중 (뼈대 유지, 표현 확장) | **상**, 문학적 |
| Gemini 3.1 Pro | 날렵하고 차가운 도시의 곡선 | 반항적 아우라 / 퓨처리스틱 고딕 | 속내 감춘 은밀한 응시 | 네온 도시 / 액체 금속 미로 / 블랙홀 | **낮음** (형식까지 이탈, 해시태그화) | 상, 강렬·과장 |

### 관통하는 공통점
- **다섯 모델 모두** '무한 거울 + 금속/유리/미로 + 무심한 냉정함'으로 수렴.
- 얼굴형은 전원 '타원형/매끈한 선'으로 일치 → 얼굴 자체 해석은 모델 간 이견이 거의 없음.

### 갈라지는 지점
- **질감 어휘**: GPT-5.6=크롬(금속), GPT-5.5·Fable 5=유리, Opus=액체 금속+초록빛. 같은 배경을 서로 다른 물성으로 번역.
- **빛의 포착**: Fable 5만 '수직으로 쏟아지는 빛'을 핵심 모티프로 잡음 → 실제 사진의 세로 조명 스트립을 가장 직접적으로 읽어냄. 다른 모델은 이 빛을 '금속/거울'로 추상화.
- **일관성↔시적임은 트레이드오프가 아니었다**: GPT 계열에선 안정(5.6)↔풍부(5.5)가 반비례했지만, **Fable 5는 둘을 동시에** 달성(2회 모두 안정적이면서 가장 문학적). 즉 트레이드오프는 모델의 한계이지 법칙이 아님.
- **형식 안정성**: Gemini가 2회차에서 '한 줄 문장' 지시를 벗어나 해시태그로 이탈 → 프롬프트 구속력이 모델마다 다름.

## 4. 배운 것 — 그래서 무엇을 알게 됐나

1. **AI는 '사람'만큼이나 '공간'을 읽었다.** 다섯 모델의 초현실 키워드가 전부 배경(미러룸) 쪽으로 쏠렸다. 즉 이 사진에서 AI가 준 해석의 상당 부분은 인물이 아니라 환경에서 왔다. → 개인화라고 부르려면 '배경 영향'을 분리해야 한다는 걸 처음으로 확인.
2. **얼굴형 해석은 합의되지만, 무드·질감은 모델의 개성이 갈린다.** 개인화의 여지는 '사실(얼굴형)'이 아니라 '해석(질감·장면)'에 있다.
3. **'일관성 vs 시적임'은 법칙이 아니라 모델의 성능 문제였다.** GPT 계열만 봤을 땐 안정↔풍부가 반비례해 트레이드오프처럼 보였으나, Fable 5가 둘을 동시에 달성하며 그 가설을 깼다. → '근거 있는 시'는 후처리 트릭 없이 **모델 선택만으로** 가능할 수 있다. 매장용 모델을 고를 때 이 두 축(재현성·표현력)을 함께 만족하는지가 핵심 기준.
4. **effort를 최대로 줘도 모델 간 편차는 그대로였다.** 조건을 맞춰도 Gemini의 형식 이탈, GPT 버전 간 차이는 남았다. → 결과 차이는 '설정'이 아니라 '모델 자체'에서 온다는 방증.
5. **빛을 본 모델이 공간을 가장 정확히 읽었다.** 실제 사진엔 세로 조명 스트립이 강한데, 이를 '빛'으로 직접 포착한 건 Fable 5뿐이었다. 나머지는 같은 요소를 '금속·거울'로 추상화. → AI마다 '무엇을 실제로 보고, 무엇을 관습적으로 채우는지'가 다르다.

## 5. 다음 결정 — 이 결과가 다음 실험을 어떻게 바꾼다

- **exp01b(즉시 후속)**: 같은 사진에 "배경은 무시하고 인물만 해석" 조건을 추가해 재실험 → '공간 영향'을 분리 측정. RQ1의 핵심 질문이 됨.
- **exp02(이미지 생성)로 넘길 재료**: 질감이 갈린 방향(크롬 / 유리 / 초록빛 금속 / 빛의 기둥)을 각각 이미지 생성 프롬프트로 만들어 비교. "해석의 차이가 그림의 차이로 이어지는가"를 봄.
- **해석 담당 모델 후보**: 현재까지 재현성+표현력을 함께 만족한 **Fable 5**가 '거울의 목소리(해석 문장)' 생성 유력 후보. exp01b·후속 사진에서도 이 우위가 유지되는지 확인.
- **프롬프트 개선**: '한 줄 문장' 형식을 지키게 하려면 지시를 더 강하게(예: "반드시 완결된 한 문장, 해시태그 금지"). Gemini 이탈이 근거.
