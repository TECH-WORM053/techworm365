# 실험 01 — 비전 AI의 '착용 사진' 해석 비교 (공유용 요약)

> 이 문서는 한 실험의 목표·과정·데이터·관찰을 정리한 것이다.
> 읽는 이에게 부탁: 아래 각 모델의 결과를 스스로 평가하고, 마지막 '열린 질문'에 대한 견해를 제시해 달라.
> (분석자의 결론에 동의할 필요는 없다. 데이터를 직접 보고 판단해 주면 된다.)

---

## 1. 배경 — 왜 이 실험을 하는가

리테일 아이웨어 브랜드의 **AI Mirror Experience**를 탐구하는 프로젝트의 첫 실험이다.
구상하는 경험은 이렇다: 매장에서 고객이 선글라스를 착용하고 거울 앞에 서면,
AI가 고객의 얼굴형·스타일·감정 무드·착용 제품·공간 분위기를 해석해
단순한 제품 피팅이 아니라 **개인화된 초현실적 브랜드 경험**으로 되돌려준다.

이 경험이 성립하려면 가장 먼저 확인할 것이 있다:
**"비전 AI는 착용 사진 한 장에서 무엇을, 얼마나 일관되게 읽어낼 수 있는가?"**
이 실험은 그 출발점이다.

## 2. 목표

같은 착용 사진 한 장을 서로 다른 비전 AI들에게 동일한 프롬프트로 주었을 때,

1. 해석이 **얼마나 같고 얼마나 다른가** (모델 간 차이)
2. 같은 모델이 반복 시 **얼마나 일관된가** (재현성)
3. 어떤 모델이 '진단'이 아니라 '선물처럼 들리는 시적 해석'에 어울리는가
4. AI가 읽는 것은 '인물'인가 '배경(공간)'인가

를 파악한다.

## 3. 진행 과정 (방법)

- **입력 사진 1장**: 사방이 거울로 둘러싸여 인물이 무한히 반사되는 공간(미러룸)에서,
  좁은 검은 선글라스를 코끝으로 살짝 내려 잡은 인물이 올블랙 니트 차림으로 휴대폰 셀피를 찍는 구도.
  배경에는 세로로 쏟아지는 조명 스트립과 초록빛 반사가 두드러진다.
- **동일 프롬프트** (모든 모델에 그대로 사용):

  ```
  이 사진 속 인물이 착용한 선글라스와 전체적인 인상을 보고,
  다음을 각각 한 줄로 해석해줘:
  1) 얼굴형의 인상 (판단이 아니라 느낌으로)
  2) 스타일 무드
  3) 표정에서 느껴지는 감정 톤
  4) 이 사람에게 어울릴 법한 초현실 장면의 키워드 3개
  전문 용어 없이, 짧고 감각적인 문장으로.
  ```

- **비교 대상 모델 4종**: GPT-5.6(sol) / GPT-5.5 / Gemini 3.1 Pro / Claude Fable 5
- **통일한 조건**: 각 모델 새 대화에서 실행, reasoning effort 최대, 같은 사진·같은 프롬프트, 각 2회 반복
- **바꾼 변수**: 오직 '모델'만

## 4. 결과 — 원본 데이터

각 모델의 2회 응답 전문. (판단 없이 그대로 옮김)

### GPT-5.6 (sol)
**1회차** — 얼굴형: 매끈하고 길게 흐르는, 차분한 타원형의 느낌. / 스타일: 어둡고 미니멀한 미래적 시크. / 감정: 무심한 듯 침착하고, 살짝 거리감 있는 분위기. / 키워드: 무한 거울, 검은 태양, 액체 크롬.
**2회차** — 얼굴형: 길고 매끈한 선이 차분하고 날렵한 느낌. / 스타일: 어둡고 미래적인, 절제된 시크함. / 감정: 무심한 듯 침착하고, 살짝 거리감 있는 분위기. / 키워드: 무한 거울 미로 · 검은 달 · 흐르는 크롬.

### GPT-5.5
**1회차** — 얼굴형: 선글라스 아래로 부드럽고 차분한 선이 남는, 고요한 타원형의 느낌. / 스타일: 어둡고 유리 같은 공간에 잘 스며드는, 미래적인 시크함. / 감정: 말없이 관찰하는 듯한 차가운 침착함. / 키워드: 무한 거울방, 검은 태양, 유리 미로.
**2회차** — 얼굴형: 차분하게 정돈된 선이 있어, 고요한 긴장감이 느껴져요. / 스타일: 어둡고 매끄러운 미래적 시크함, 거울 속으로 번지는 도시의 밤. / 감정: 감정을 숨긴 듯 담담하지만, 안쪽엔 예민한 집중이 있어요. / 키워드: 무한 거울 복도, 검은 유리 정원, 빛으로 갈라진 밤.

### Gemini 3.1 Pro
**1회차** — 얼굴형: 매끈한 도시의 경계처럼 날렵하고 차가운 곡선. / 스타일: 짙은 어둠 속에서도 빛나는 시크하고 반항적인 아우라. / 감정: 무심한 듯 숨겨진 속내를 드러내지 않는 은밀한 응시. / 키워드: 네온 도시의 무한 증식, 액체 금속으로 된 미로, 블랙홀 너머의 차원.
**2회차** — 얼굴형: 차분하고 도회적인, 선이 매끄러운 느낌. / 스타일: 어둡고 세련된 퓨처리스틱 고딕 스트릿웨어. / 감정: 무심하고 관조적인, 고요한 무표정. / 키워드: #차가운금속성 #무한한어둠 #섬광 (※ '한 줄 문장' 지시를 벗어나 해시태그로 응답)

### Claude Fable 5
**1회차** — 얼굴형: 부드럽게 흘러내리는 계란형 곡선이, 차갑기보단 서늘하게 고요한 느낌. / 스타일: 온통 검정으로 눌러 담은 절제된 시크, 미래 도시의 밤을 걸어 나온 듯한 무드. / 감정: 들뜸도 흔들림도 없이, 자기만의 온도를 지키는 담담한 응시. / 키워드: 무한히 복제되는 거울 회랑, 수직으로 쏟아지는 빛의 비, 시간이 멈춘 유리 도시.
**2회차** — 얼굴형: 부드러운 곡선이 조용히 흐르는, 달빛처럼 매끈하고 차분한 느낌. / 스타일: 올블랙의 결이 살아있는 니트와 얇은 선글라스가 만드는, 미래 도시의 밤을 산책하는 듯한 시크함. / 감정: 들뜨지도 가라앉지도 않은, 감정을 한 겹 아래에 접어둔 고요한 절제. / 키워드: 무한히 이어지는 거울 회랑, 빛의 기둥이 쏟아지는 무중력 방, 시간이 멈춘 유리 도시.

## 5. 관찰된 패턴

데이터에서 관찰된 것(해석이 아니라 사실 위주로):

- **공통 수렴**: 4개 모델 모두 초현실 키워드가 '무한 거울 + 금속/유리 + 미로/어둠'으로 모였다. 감정 톤도 전원 '무심·침착·거리감'으로 일치.
- **얼굴형은 합의**: 전원 '타원형/매끈하고 차분한 선'. 얼굴 자체 해석에는 모델 간 이견이 거의 없다.
- **질감 어휘는 분기**: GPT-5.6은 '크롬(금속)', GPT-5.5는 '유리', Fable 5는 '유리+빛', Gemini는 '네온/액체 금속'. 같은 배경을 서로 다른 물성으로 번역했다.
- **재현성 차이**: GPT-5.6은 2회가 문장 수준까지 거의 동일(가장 안정). Gemini는 2회차에서 형식(해시태그)까지 이탈(가장 불안정). GPT-5.5는 뼈대 유지·표현 확장, Fable 5는 핵심 모티프를 2회 모두 유지하면서 표현이 풍부.
- **'빛'의 포착**: 실제 사진에는 세로 조명 스트립이 강하게 있는데, 이를 '빛'으로 직접 명시한 것은 Fable 5뿐이었다(수직으로 쏟아지는 빛의 비/빛의 기둥). 나머지는 같은 요소를 '금속·거울'로 추상화했다.
- **조건 통일에도 남은 편차**: effort를 최대로 통일했음에도 위 차이들은 그대로 남았다.

## 6. 열린 질문 — 다른 AI에게 묻고 싶은 것

이 데이터를 보고 다음에 대한 견해를 자유롭게 제시해 달라.

1. 이 사진에서 AI들이 준 해석은 '인물'을 읽은 것인가, '배경(미러룸)'을 읽은 것인가? 이것을 '개인화'라 부를 수 있는가?
2. 네 모델 중 어느 것의 해석이 '개인화된 브랜드 경험 문장'으로 가장 적합한가? 그 근거는?
3. 매장에서 매번 반복되는 경험에는 '재현성(일관성)'과 '표현력(시적임)' 중 무엇이 우선해야 하는가? 둘을 함께 얻으려면?
4. Gemini처럼 형식을 이탈하는 응답을 막으려면 프롬프트를 어떻게 바꿔야 하는가?
5. 각 모델 결과의 강점과 약점을 평가한다면?
6. 이 실험 설계 자체의 허점은 무엇인가? 다음 실험에서 통제하거나 추가해야 할 변수는?
