# Experiments — 실험 아카이브

세 프로젝트의 실험을 기록하는 공간입니다.
각 실험은 방(폴더) 하나씩 차지하며, 안에 로그(README)와 결과물이 함께 들어갑니다.

## 폴더 구조

```
experiments/
  exp01-비전모델-해석비교/
    README.md      ← 실험 로그 (질문→방법→결과→배운 것→다음 결정)
    inputs/        ← 실험에 넣은 사진·프롬프트
    outputs/       ← 각 AI 도구가 만든 결과물
    assets/        ← 비교 스크린샷·정리 이미지
```

## 작업 루프

1. 외부 도구로 결과물을 만든다
2. 해당 실험의 `outputs/`에 저장한다
3. Claude에게 폴더/파일을 알려준다
4. Claude가 이미지·텍스트를 직접 보고 피드백한다
5. 함께 로그(README)를 채운다
6. git에 커밋한다 (커밋 하나하나가 탐구 과정의 기록)
