# Research Notes — Mirror Boundary Lab v0.2

## 출발점

실험 01의 비전 모델 결과는 서로 다른 모델임에도 `무한 거울`, `금속/유리`, `어둠`, `거리감`으로 수렴했습니다. 이는 모델이 인물 고유의 특성보다 사진의 강한 배경 문법을 읽었을 가능성을 보여줍니다.

따라서 이 프로토타입은 “AI가 나를 정확히 읽는가?” 대신 다음을 묻습니다.

> AI 해석이 불완전하다는 사실을 숨기지 않고 사용자가 해석 범위를 조절하게 하면, 그 경험은 오히려 더 개인적이고 편안해지는가?

## 이번 버전의 연구 질문

- RQ1. 사용자는 AI가 자동으로 고른 문장보다 자신이 선택한 문장을 더 개인적으로 느끼는가?
- RQ2. `낯섦의 거리`를 직접 조절하는 행위가 결과에 대한 통제감을 주는가?
- RQ3. 얼굴형·감정 자동 판정을 제거하면 신기함이 줄어드는가, 불편함이 더 크게 줄어드는가?
- RQ4. 로컬 처리와 즉시 삭제 안내를 사용자가 실제로 이해하고 신뢰하는가?

## 5분 사용자 테스트

참여자에게 기능을 설명하지 않고 링크만 전달합니다.

1. 동의 → 프레임 선택 → 문장 선택 → 장면 조절 → 평가까지 혼자 완료하게 합니다.
2. 완료 시간을 기록합니다. 목표는 60초, 허용 범위는 90초입니다.
3. 결과 화면의 두 5점 평가를 받습니다.
4. 아래 질문을 짧게 묻습니다.
   - 어느 순간 “내가 만든 결과”라고 느꼈나요?
   - 어느 문구나 화면이 가장 불편하거나 이해하기 어려웠나요?
   - 슬라이더를 움직일 때 무엇이 변한다고 느꼈나요?
   - AI가 얼굴을 자동 분석하지 않는다는 점이 아쉬웠나요, 안심됐나요?
5. 참가자가 저장한 JSON을 받아 익명으로 표에 합칩니다. 원본 이미지는 받지 않습니다.

## 판단 기준

| 지표 | 첫 목표 | 다음 결정 |
|---|---:|---|
| 설명 없이 완주 | 5명 중 4명 이상 | 미달 시 화면 수보다 문구와 선택 구조부터 수정 |
| 중앙 완료 시간 | 90초 이하 | 초과 시 해석 카드 또는 물성 선택을 축소 |
| 개인화 체감 | 평균 3.5/5 이상 | 낮으면 시각 신호와 문장의 연결을 강화 |
| 얼굴 사용 불편함 | 평균 2.5/5 이하 | 높으면 기본값을 합성 샘플로 바꾸고 동의 문구 재검증 |
| 낯섦 조절 인지 | 5명 중 4명 이상 | 미달 시 슬라이더 변화 폭과 레이블을 키움 |

## 현재 구현이 주장하지 않는 것

- 이 장면이 사용자의 감정이나 정체성을 정확히 표현한다.
- Canvas 효과가 실제 생성형 이미지 모델의 품질을 대표한다.
- 로컬 신호 측정이 AI 얼굴 분석이다.
- 특정 브랜드의 공식 경험 또는 시각 시스템이다.
